# Fleet Vision: Tier 2 Intelligence & Safety

Transitioning from "Mobile Manipulation" to **Autonomous Domestic Agency** (SOTA v16.16).

## 🎯 1. Spatial Intelligence: 3D Semantic SLAM
We are moving beyond 2D LiDAR point-clouds. The apartment is now a **Digital Twin**.
- **Niantic Splatting**: Using the **Niantic/Scaniverse** ecosystem for room-scale Gaussian Splatting. These photorealistic maps provide the basis for semantic object identification.
- **Spatial RAG**: The fleet utilizes RAG to query the 3D map for specific object locations and hazard status ("Where is the chefs knife held in the 3D scan?").

## 🛡️ 2. Domestic Hazard Mitigation (The Redline Standard)
To ensure the safety of the household, Tier 2 implements advanced failsafes for "Fluid & Flame" manipulation:
- **Thermal Monitoring**: Continuous IR/Thermal sensing during ANY task involving high-heat (Boiling water, Gas stove).
- **Adaptive Time-outs**: Hard-coded safety durations to prevent bath-overflow or kettle dry-boil scenarios.
- **Dangerous Goods Exclusion**: Robots are physically and logically barred from manipulating lethal tools (Blades, Caustics) unless in a restricted "Chef/Maintenance" sub-state.

## 🛒 3. External Mobility: The SPAR Mission
Bumi’s utility extends beyond the front door via the **External Logistics Protocol**:
- **The Passive Trolley**: Using a standard, low-cost push/pull shopper (towed or pushed by Bumi).
- **Prepaid Payment**: Standardized NFC/QR payment integration mounted on the chassis for cashier-neutral checkouts.
- **Social Bounds**: External missions are limited to moderate grocery loads and 100% sidewalk-safe navigation.

## 🏛️ 4. Economic Pragmatism
Tier 2 rejects "Total Robotification." Every automation must be justified by:
1. **Safety Enhancement** (e.g., bath-overflow protection).
2. **True Time-Savings** (e.g., grocery delivery).
3. **Hardware Reuse** (making "dumb" objects robot-enabled via AR tags).
